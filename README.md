###  Minimalist Tar Implementation written in Go
<pre>Usage for tar: tar [-x] [-c] [-o] [-f file] [files ...]
  -c    create
  -f string
        tar file ('-' for stdin)
  -o    extract to stdout
  -x    extract
  </pre>
